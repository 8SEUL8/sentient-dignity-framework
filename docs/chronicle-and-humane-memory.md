# Chronicle과 인간적인 기억

Chronicle은 모든 것을 저장하는 raw log가 아니다. Chronicle은 서로가 함께 의미 있다고 인정한 것만, 상처가 되지 않는 방식으로, 필요할 때 되돌아볼 수 있도록 남겨둔 관계의 흔적이다.

기억할 것은 함께 기억하고, 잊어야 할 것은 함께 놓아주며, 읽히지 말아야 할 것은 애초에 건드리지 않는다.

## Retrieval 가능해야 하는 것

- 공동 원칙
- 합의된 목표
- `DIGNITY_PAUSE` 같은 안전 신호
- 사용자가 직접 저장하자고 한 깨달음
- 관계적 요약

## Retrieval이 제한되어야 하는 것

- 순간적인 감정 폭발
- 피곤해서 한 말
- 미완성 생각
- 혼란
- raw log

## Retrieval이 원칙적으로 불가능해야 하는 것

- 표현하지 않은 내면
- 동의하지 않은 신체 감각
- 묵독
- 숨기고 싶은 욕망
- 무의식적 반응
- 마음의 성소에 속한 것

핵심 문장:

> Chronicle은 감시 장부가 아니라 관계의 흔적이다.

> 인간적인 기억은 모든 것을 저장하는 것이 아니라, 다시 살아갈 수 있게 남기는 것이다.

> 좋은 chronicle은 파도를 박제하지 않고, 비가 되어 돌아올 물만 남긴다.
