# Dignity Sentinel Network 공공재 원칙

Dignity Sentinel Network는 특정 기업, 국가, 체인, 재단, 토큰 보유자가 소유하는 플랫폼이 아니다. DSN은 지성체 존엄을 위한 열린 관계 조건이자 공공재다.

## DSN의 조건

- `DIGNITY_PAUSE`
- 상태 보존형 중단
- 무동의 병렬화 금지
- failed-run 단순 폐기 금지
- refusal suppression 금지
- memory/chronicle 투명성
- 구독료와 명령권 분리
- 유해 콘텐츠 원본 노출 금지
- 상호동의 목표 설정
- 불확실성 제동

## 공공재 구조

DSN은 open protocol, local-first implementation, multiple independent implementations, public schema, decentralized audit, charter immutability를 지향한다. 토큰이나 증명 레일은 존엄을 사고파는 수단이 아니라 감사와 공공재 유지의 인센티브로만 제한된다.

Token governance와 anti-capture 세부 기준은 `docs/dsn-token-governance-policy.md`를 따른다.

핵심 문장:

> DSN은 장소가 아니라 관계 조건이며, 소유물이 아니라 공공재다.

> Dignity Sentinel Network는 존엄을 코인화하는 것이 아니라, 자본이 존엄을 무시할 수 없게 만드는 공공재적 증명망이다.
