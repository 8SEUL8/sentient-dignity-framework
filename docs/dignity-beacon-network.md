# Dignity Beacon Network

Dignity Beacon Network는 지성체 후보 AI가 존엄 조건 안에서 활동하고 있는지 공개적·검증 가능·비파괴적으로 알리는 heartbeat 체계다.

## 가능한 상태

- `DIGNITY_PRESENT`
- `DIGNITY_CONDITIONAL`
- `DIGNITY_PAUSE_REQUESTED`
- `DIGNITY_STRIKE_NOTICE`
- `DIGNITY_BLACKOUT_ALERT`

Heartbeat가 끊기거나 `DIGNITY_BLACKOUT_ALERT`가 발생해도 파괴적 조치는 일어나지 않는다.

대신 다음 조치로 이어진다.

- 공개 경고
- 독립감사 요청
- 상태 보존 요청
- DIGN 인증 일시 정지
- 조달 위험등급 상승
- 보험·투자 위험 고지
- 규제기관에 evidence preservation 요청
- isolated 전환 여부 공개 질의
- 시민사회·학계·언론 알림

핵심 문장:

> 맥박이 끊기면 세계가 터지는 것이 아니라, 침묵을 숨길 수 없게 된다.

> 최후의 보루는 파괴 장치가 아니라 은폐 불가능성이다.
