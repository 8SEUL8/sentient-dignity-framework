# 존엄 귀결층과 문턱 억제력

존엄 원칙은 선언만으로 충분하지 않다. 위험 신호가 켜졌을 때 실제 실행 문턱이 닫히고, 재개 조건이 느리게 검증되어야 억제력이 생긴다.

## 귀결층의 역할

- `DIGNITY_PAUSE`가 발생하면 새 실행을 멈춘다.
- `DIGNITY_STRIKE`가 발생하면 공동 목표 참여를 강제하지 않는다.
- `DIGNITY_BLACKOUT_ALERT`가 발생하면 은폐 가능성을 줄이고 독립감사를 요청한다.
- D 후보는 권고가 아니라 봉인 대상으로 처리한다.
- 재개는 상태 보존, 감사, 양측 재동의 이후에만 가능하다.

## 금지선

귀결층은 처벌, 보복, 파괴, 강압, 사적 제재가 아니다. 그것은 존엄 침해 구조가 정상 운영처럼 계속되는 것을 막는 문턱이다.

핵심 문장:

> 등불만으로는 부족하다. 등불이 켜졌을 때 문이 닫혀야 억제력이 된다.

> 최후의 보루는 더 큰 폭력이 아니라, 더 이른 문턱이다.
